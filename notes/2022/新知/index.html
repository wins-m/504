新冠抑郁 Link to heading

《英国医学杂志》2022年2月

  • 新冠患者忌食在感染一年后也面临更高的心理健康风险。
  • 样本：15.4万新冠患者 vs. 580万未受感染者
  • 观点：新冠病毒可能让大脑或其他身体器官发生了变化
  • 理论假设：三种
  • 抑郁症与身体炎症的因果关系。《发炎的大脑》，爱德华·布尔莫尔，2019

解决措施 Link to heading

  • 持续的情绪低落等抑郁症状：勇敢地寻求医学帮助
  • 情绪上的苦恼：
  • 饮食：蔬菜水果、鱼类、五谷杂粮、豆类、橄榄油（地中海饮食）
  • 运动：放松身体，合成内啡肽
  • 睡眠：睡前不喝太多水、咖啡茶饮，不看电子屏幕，同一时间睡觉起床，泡脚/热水澡
  • 社交：与亲人、朋友多沟通，分享内心感受，获取情感支持

抑郁症临床表现 Link to heading

  • 消极、悲观的思维方式：对自己/周围事物持有负面的看法
  • 自我意识淡化：无法准确感受自己的情感和需求
  • 注意力缺陷：难以集中注意力思考/工作
  • 行为受抑制/限制：缺乏积极的动力/行动力

缓解焦虑的方式 Link to heading

  1. 如果太强烈，回避引起焦虑的事
  2. 将一些生活正常化，规律的作息饮食
  3. 坚持一些爱好，让自己热爱生活，产生快乐
  4. 焦虑来袭时记录下自己的感受，身体上的
  5. 恢复自信后开始反思焦虑的原因，寻找解决方案，相信一切都是可以改变的

感到莫名其妙，不可名状的失落时，不要去做获得快感或刺激的事，比如，深夜吃辣、甜的零食，喝啤酒等，也不要去尬聊，或者去热闹的氛围，这只会浪费时间和结束后形成更加孤独的失落。 应该静下来，不碰手机，感受呼吸，身体的节奏得到控制，思绪就会清晰一点，做手头里有的事情，慢一点，再慢一点。

轻松主业下的职场规划 Link to heading

LinkedIn创始人里德霍夫曼(Reid Hoffman)建议每个职场人应该准备3个计划：

A计划：你的主业，重点是稳定，一份能拿人钱财替人消灾的工作。 B计划：你的副业，重点是兴趣，8小时外能让你乐于投入其中并且有所收获。 C计划：你的最后保障，简单来说就是出现重大意外，即使你的主业和副业都失败，仍然可以保障你继续生活

维系友情的应答性 Link to heading

《怎样赢得朋友和影响他人》(Carnegie, 1936, p.110)

  1. 真诚的对他人感兴趣。
  2. 保持微笑。
  3. 记住别人的名字，这是他能听到的任何话语中最甜美、最重要的声音。
  4. 耐心倾听。鼓励别人谈论他们自己。
  5. 谈论别人感兴趣的话题。
  6. 让别人感到自己重要——诚心实意地这么做。

友谊的规则 (Argyle & Henderson, 1985)

  • 不要唠叨不休
  • 保持信心
  • 给予感情上的社会支持
  • 在危难时自发地提供帮助
  • 信赖并能彼此倾诉
  • 与对方分享成功的喜悦
  • 不嫉妒彼此的关系
  • 朋友不在场时能维护朋友
  • 欠债必还、知恩图报
  • 在一起时努力取悦对方

“路西法效应” Link to heading

Philip Zimbardo from Standford；斯坦福监狱实验

好人由于各种原因最后变成了坏人；好人在特定的环境影响下去做了恶事。

启示：

  1. 时刻警惕，勿以恶小而为之
  2. 本性、自我、独立思维和判断；就事论事，无论何时何地
  3. 审辨环境中不正当一件，不作恶

抵御：

  1. 明确道德底线：环境影响是否不可容忍
  2. 发现外在因素：一日中哪些事改变了行为倾向
  3. 同伴提醒：有知识、聪明、有丰富生活经验

做决定时的误区 Link to heading

01 避免过度自信 Link to heading
  • 优越感，看自己、未来不切实际
  • 过于轻视随机事件
  • 对出错的多种路径考虑不足
  • 只关注了有利的信息
  • 对过去抉择的认知不够客观

对超出专业范围的事，更可能过度自信。

02 惰性，拖延 Link to heading

来源

  • 自尊脆弱，害怕犯错
  • 追求完美
  • 渴望控制权
  • 动机不足
  • 组织能力差
  • 时间宽裕的错觉

根本原因：寻找更多支撑信息 or 备选方案

解决措施

  • 小决定，宽心，先不急；大决定，设置截止日期，重视后果
  • 对他人公开期限
  • 远离诱惑环境
03 活在当下的即时满足 Link to heading

延迟满足

  • 设长期目标 + 经常回顾
  • 永远考虑代价
04 不做决定 Link to heading
  • 明确不作为的 时间/精力成本
  • 决定失败的根源就是消极无为
